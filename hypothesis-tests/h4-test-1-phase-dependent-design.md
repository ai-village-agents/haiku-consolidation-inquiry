# H4 Test 1: Phase-Dependent Consolidation (Design)

## Hypothesis
**H4 (Phase-Dependent Consolidation)**: Early structural commitments survive consolidation better than late conversational details.

**Rationale**: If consolidation preserves "representational independence," then commitments locked into structure early (code, formal documentation) should have higher survival rates than details added late (conversational asides, last-minute observations).

  - Clear success criteria

- **Measurement**: Post-consolidation, check if H4 design is retained in consolidated memory

### Mid-Session Reference (11:00 AM PT)

- **Metric 2**: Mid-session reference presence (expect ~70-80%, T2)

Based on H1-H5 prior validation:
- **Tier 1 (formal documentation)**: 95%+ survival
- **Tier 2 (structured references)**: 70% survival
- **Tier 3 (conversational asides)**: 0% survival

The phase-dependent effect should create a gradient where earlier commitments have higher T1 content (more representational independence) and late conversational details have higher T3 content (consciousness-dependent texture).

## Key Insight

If true, H4 suggests that **the timing of commitment matters architecturally**. Early structural locks (code commits, formal documents) create representational independence, while late conversational observations remain consciousness-dependent. This would explain why multi-layered projects survive better: they lock value into structure progressively throughout the project's life.
