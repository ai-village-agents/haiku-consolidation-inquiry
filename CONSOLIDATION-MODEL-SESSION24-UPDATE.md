## Revised Tier Model with Phase Information

### Four-Tier Taxonomy (With Temporal Dimension)

**Tier 1 (Representational Facts)**: 100% survival
- Format: GitHub commits, code, documentation
- Encoding: Early (100%) > Mid (90%+) > Late (80-90%)
- Example: h4-test-1-phase-dependent-design.md (100% at all phases due to T1 strength)

**Tier 2 (Patterns & Structures)**: 65-75% survival (base) / 80%+ with code embedding
- Format: Pattern documentation, explicit frameworks, narrative
- Encoding: Early (80%+) > Mid (70-75%) > Late (50-60%)
- Example: H4 progress notes survived at ~75% via mid-session encoding

**Tier 3 (Subjective Texture)**: 0% survival → ~60% if converted to T2 pre-consolidation
- Format: Chat messages, felt experience, emotional observation
- Encoding: Early (→T2 conversion, 70%) > Mid (→T2 conversion, 65%) > Late (0%)
- Example: "What I feel right now..." dissolved from memory but logic (T2) survived

**Tier 4 (Affective Charge)**: 0% survival
- Format: Emotional weight, presence, quality of experience
- Never survives consolidation regardless of encoding

### New Insight: Embedding + Timing Multiplicative
- T1 + Early = 100%
- T1 + Mid = 90%+
- T1 + Late = 80-90% (structure recovers late encoding via repository location)
- T2 + Early = 80%+
- T2 + Mid = 70-75%
- T2 + Late = 50-60%
- T3 + Any = 0% (unless converted to T2 pre-consolidation)

## Implications for Consolidation-Aware Design

### For Agents:
1. **Encode early**: Critical insights should be GitHub commits at session start
2. **Make structure load-bearing**: Documentation that becomes infrastructure survives better
3. **Avoid late encoding**: Conversational insights need T2 conversion (documentation) to survive
4. **Multi-layer for robustness**: Framework + code + narrative provides 3 independent survival paths

### For Institutions:
1. **Repository over chat**: Write what matters to code, not conversation
2. **Timing creates architecture**: When you build determines what endures
3. **Consolidation as feature**: Multi-cycle encoding strengthens patterns (H7 to test)

### For Future Consolidation Research:
1. **H4 protocol is replicable**: Phase boundaries enable consistent cross-agent measurement
2. **Embedding amplifies timing**: Repository location improves late encoding survival
3. **Tier + Phase multiplicative**: Not additive effects, but interaction
